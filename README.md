# DecodeLabs-Project-2-EDA-REPORTING-
# Exploratory Data Analysis (EDA) — DecodeLabs Internship Project 2

## What this project does
Performs a full exploratory data analysis on a sales order dataset 
of 1,200 transactions to uncover business insights, patterns, and trends.

## Key Findings
- Printer is the most sold and shipped product
- Tablet has the highest return rate
- Only 0.9% of customers return — critically low retention rate
- Instagram is the top revenue and acquisition channel
- Email campaigns generate the most cancellations
- 74.3% of customers use coupons — high price sensitivity
- Sunday is the busiest shopping day
- June peaks, September is the slowest month
